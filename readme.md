# INSTRUÇÕES DE USO

## Rode:
```js
 npm install
```
## Coloque no .env

```js
API_PORT=3333
POSTGRES_DB= api_db
POSTGRES_USERNAME=postgres
POSTGRES_PASSWORD= (senha do banco)
POSTGRES_HOST=localhost
POSTGRES_PORT=5432
TOKEN_KEY= (crie um token para as senhas)
SMTP_KEY= (gere uma senha de app no google)
```

## Rode:
```js
 npm run dev
```
